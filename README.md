# MyAnimeList Companion

A lightweight Flutter companion app for MyAnimeList (MAL) that provides fast access to essential anime information through a clean, minimal interface. The app combines core MAL content with Firebase-powered community features for anime fans to interact and share.

## üöÄ Getting Started

*Note: This is a showcase project. For a production version, the following setup would be required:*

1. **Clone the repository**
   ```bash
   git clone https://github.com/Amr3006/Mal_App.git
   cd Mal_App
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Configure Firebase**
   - Create a Firebase project
   - Add iOS and Android apps to your Firebase project
   - Download configuration files and add to the project

4. **Run the app**
   ```bash
   flutter run
   ```
